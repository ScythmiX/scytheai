ScytheAI is a conceptual interface for an AI productivity tool designed to demonstrate practical frontend engineering skills. The project integrates a multi-section UI, reactive components, and a custom 3D environment. It is intended as a professional portfolio piece rather than a production application.

The central visual element is a Three.js background scene featuring glowing square elements that subtly track the user's scroll position. This interaction produces a clean, modern motion effect without distracting from the interface. The foreground UI is built with React and TailwindCSS, using modular components and structured data pipelines to preserve clarity and maintainability.

The project reflects real-world frontend workflows: a separate rendering layer, component-driven UI, reusable widget-level elements, and a clear directory hierarchy.

---

## Features

- Scroll-reactive Three.js environment with glowing square elements
- Responsive React UI with well-defined sections (Features, Pricing, Testimonials, etc.)
- Modular components and structured data files
- Logical separation between UI, rendering, and utility logic
- Modern, polished presentation suitable for client inspection and demonstration

## Usage

1. Clone the repository.

2. Install dependencies:
`npm install`

3. Start development server:
`npm run dev`

4. Visit the provided local URL to view the project.


- To build for deployment:
`npm run build`
